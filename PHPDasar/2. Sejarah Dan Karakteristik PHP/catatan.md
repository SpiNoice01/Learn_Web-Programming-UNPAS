Karakteristik Bahasa PHP
========================

1. Ekstensi file harus `.php` (walaupun isinya HTML)
2. Ditulis dalam tag PHP (Delimiter PHP `<?php ?>`)
3. Bisa digunakan bersamaan dengan HTML
4. Mengikuti sintaks bahasa C
